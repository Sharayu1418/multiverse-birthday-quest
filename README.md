# Multiverse Birthday Quest

A birthday adventure game where you travel through beloved fictional worlds and solve puzzles to restore the multiverse.

## Worlds

- **Marvel Universe** — Superhero-themed puzzle
- **Harry Potter World** — Marauder's Map maze with enemies, letter collection, and the Fat Lady's password
- **Percy Jackson** — Olympus-themed challenge
- **Friends** — Central Perk puzzle
- **Stranger Things** — Hawkins signal adventure

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:8080](http://localhost:8080).
